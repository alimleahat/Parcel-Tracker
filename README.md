# Package Delivery Tracker (CLI â€“ C)

A command-line based package delivery tracking system written in C.  
Developed as part of the **ELEC2645 Individual Project**, this application allows users to manage and track multiple parcel deliveries from different couriers within a single interface.

---

## Features

- Add and manage parcel delivery orders
- Persistent storage using text files
- Automatic delivery status updates based on time
- View current and delivered orders separately
- Search orders by package ID
- Sort delivered orders by most recent delivery
- Delivery cost calculation based on courier data
- Basic statistics and visual summary (bar chart)
- Unit tests for key functionality

---

## How to Build and Run



### Build & Run
```bash
make
./tracker
```

### Run Tests
```bash
make tests
./tests
```
